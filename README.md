# FUTURE_CS_02 – SOC Alert Monitoring & Incident Response Simulation

## Internship Track
Cyber Security Internship

## Task Number
Task 2 – Security Alert Monitoring & Incident Response Simulation

## Objective
To simulate the role of a SOC analyst by monitoring security alerts, analysing logs, identifying threats, and responding to incidents.

## Tools Used
- Kali Linux
- Linux Logs
- iptables
- SSH Service
- MS Word (for report)

## Task Summary
- Analysed simulated SOC logs
- Identified brute force, malware, and reconnaissance alerts
- Classified incidents by severity
- Simulated incident response actions
- Documented findings in an incident response report

## Files Included
- Incident Response Report
- SOC Logs
- Alert Analysis
- Command Execution Screenshots

## Outcome
Gained hands-on understanding of SOC operations, alert triage, and incident response workflows.
